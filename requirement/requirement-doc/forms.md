## 练习题

### 练习1：识别模糊需求
请将以下模糊需求改写为明确的需求：

**模糊需求**："系统应该快速响应"

**改写后**：
```
【NFR-001】响应时间
- 普通列表查询响应时间 ≤ 1秒（P95）
- 复杂统计查询响应时间 ≤ 3秒（P95）
- 页面加载时间 ≤ 2秒（P95）
- 文件上传操作反馈时间 ≤ 500ms
```

### 练习2：编写验收标准
为以下功能编写至少5条验收标准：

**功能**：用户密码找回功能

**验收标准示例**：
```
【FR-015】密码找回功能

验收标准：
- [ ] 支持通过注册邮箱找回密码
- [ ] 找回链接有效期为30分钟
- [ ] 密码重置后自动失效旧链接
- [ ] 新密码必须包含8-20位字符
- [ ] 密码重置成功后显示确认页面
- [ ] 重置完成后自动登出其他设备
```
